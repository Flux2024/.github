  <h3>프로젝트 목표</h3>
  <p>본 프로젝트는 최신 웹 기술을 활용하여 온라인 경매 웹 애플리케이션을 구축하는 것을 목표로 합니다.</p> 
  <p>백엔드는 <strong>Spring 프레임워크</strong> 를, 프론트엔드는 <strong>Vue.js 프레임워크</strong> 를 사용하여 구현할 예정입니다.</p>
