  <h3>🚀프로젝트 목표</h3>
  본 프로젝트는 최신 웹 기술을 활용하여 온라인 경매 웹 애플리케이션을 개발하는 것을 목표로 합니다.<br>
  <br>
  백엔드는 안정성과 확장성을 갖춘<br> 
  Spring 프레임워크를 활용하여 구현되었으며,<br> 
  <br>
  프론트엔드는 사용자 친화적인<br> 
  Vue.js 프레임워크를 사용하여 구축되었습니다.<br> 
  <br>
  이를 통해 사용자에게 매끄러운 경매 경험을 제공하는 것을 목표로 합니다.<br>
  <br>
